# AI Course Planner
Enter your major, remaining requirements, interests, and workload preference to get a generated semester schedule.

**Live demo:** https://course-planner-ui.vercel.app

## Stack
FastAPI, Groq (Llama 3.3 70B), React + Vite.

## What I'd add next
- Real integration with a university's course catalog/API instead of free-text requirements
- Multi-semester planning toward a full degree audit
- Save and compare multiple generated plans (Supabase schema is in place for this)
